
Создать бота предоставляющего по запросу погоду в выбранном городе.
В качестве сервиса для работы с погодой можно использовать https://openweathermap.org/




Что нужно сделать:
Создайте бота для одной из следующих платформ:
Telegram, 
Discord
и др.

Бот обладает следующим набором функций:

- Принимает сообщения и команды от пользователя,
- Сохраняет историю сообщений пользователя,
- Позволяет пользователю просмотреть историю запросов,
- (задача со звездочкой) Позволяет скачать всю историю в выбранном формате (txt\json\xml)

Команды можно делать разные, но среди них должны присутствовать команды /start и /help

Вы можете сделать бота на любую тематику работы с любым сервисом. В качестве примера взят "погодный бот и сервис openweathermap", но можно использовать другие сервисы, например meteoservice или яндекс погоду (https://yandex.ru/dev/weather) если это возможно. 

Помимо этого можно, изменить тематику бота на 
- поиск видео на YouTube
- отправлять курс валют с заданным интервалом
- присылать интересные факты из собственного хранилища
и др.

Что оценивается
- Бот принимает текстовые сообщения.
- Бот реагирует на команду /start и /help
- В проекте более одного файла с исходным кодом
- Использование системы контроля версий
- В readme файле проекта есть понятное описание, что менялось от коммита к коммиту (намёк на поддержку версионности продукта)
- Приветствуется работа в команде

Как отправить задание на проверку
- Работу присылать ссылкой на актуальный пулреквест с последними изменениями по проекту.
